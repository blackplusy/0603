-----------------------------------------
1.复合数据类型
	数值、字符、布尔类型都是js的简单数据类型，复合数据类型有对象、数组等等
	1>常用的内置对象
	面向对象设计模式中，将数据和处理数据的方法捆绑在一块，形成一个整体成为对象
	对象封装了数据和操作数据的方法，要使用数据或者方法必须先创建一个对象，使用new来调用对象，从而创建一个对象
	var obj=new object()   创建新对象
		Array  		数组对象，封装来数组的操作和属性
		Boolean		布尔对象
		Date        日期对象，封装了和日期相关操作和属性
		Math        数学对象，提供基本的数学函数和常量
		Global      全局对象，全局函数和全局常量都为该对象所有
	2>日期对象
	日期对象一个典型的应用就是获取当前时间
	http://www.w3school.com.cn/js/jsref_obj_date.asp
	3>全局对象
	全局方法的拥有者
		isNaN()             判断值是否是NaN，返回一个布尔值
		parseInt() 			返回由字符串转换得到的整数
		parseFloat()        返回由字符串转换得到的浮点数
		注意:NaN  not a number
	4>数学对象
	封装了和数学相关的特性，包括一些常数和数学函数
	调用mMath对象的方法
	math.[{属性(方法名)}]
		PI     	返回圆周率
		abs  	返回数字的绝对值
		sin     返回数的正弦
		min     返回给定数字中最小值
		max     返回给定数字钟最大值
	5>字符串对象
	分装了和字符串相关的特性，来处理字符串，可以对字符串进行剪切、合并，替换等操作
	调用string对象的方法或者属性
	string([val])
	注意：val是可选项
	6>数组对象
	js中基本的数据类型，封装了和数组相关的方法和属性
	房间1      房间2     房间3     房间4
	范冰冰     刘亦菲    林志玲    李冰冰
	创建数组的方式
	1.通过new运算符调用Array对象构造函数
		var a=new Array();
	2.通过构造函数传递元素作为参数(创建带指定元素的数组)
		var a=new Array(10,20,30,'apple');
	3.直接将元素放入[]中，通过逗号分隔开
		var a=[1,2,3,4]
	4.创建指定上的数组(创建数组指定元素个数)
		var a=new Array(3);
	注意:放在数组后面括号中数字就是所谓索引，从0开始，可以通过索引取出来我们需要的数据，从左向右从0开始
2.其他数据类型
	1>函数
	函数的角色有2种，一种是数据类型方面，另外一种是程序设计方面
	定义函数的方式
		function typename(arg)
		{
			this.arg=arg;
		}
	2>空值
	之前所说数据类型都有自己内容，编程中还有一种：什么都没有(null)
	注意：一个=代表赋值，两个=代表比较，三个=代表对象比较
	3>不确定的数据类型
	undefined 表示一个变量什么都没有得到，连空都没有
	在应用里面null和undefined实际意义是等效的
3.数据类型转换
	1.隐式数据类型转换
	程序在运行时候，系统根据上下文需要，自动将数据转换为另外一种类型数据的过程
	2.显示数据类型转换
	通常使用parseInt(),parseFloat()

变量和常量
1.常量
从始至终不能被改变的数据
a.常量的种类
包括字符类型，布尔类型，数值类型，null等
"天儿不错！！！"     字符常量
077                 数值常量
true                布尔常量

2.变量
a.什么是变量
变量只在程序运行中可以发生值改变的量，专业说法是可读写的内存单元
可以理解为装载东西的容器，变量名代表系统分配给他们的内存单元

b.变量的定义方式
var 变量名=值;       定义变量
变量名=值;            赋值
var  变量名;         定义变量
字符类型变量
var str="hello";
布尔类型变量
var b=true;
数值类型变量
var n=10;
复合类型
var obj=new object();

c.变量的命名规则
变量名必须以字母或者下划线或者$开头，后面可以跟下划线或者数字，不能使用特殊符号
js对标识符大小写敏感

d.变量的作用范围
作用域，指有效范围，js变量作用域有全局和局部之分，变量同名时候局部作用优先于全局作用域

e.变量的用途
主要适用于存储数据，比如计算结果，存储用户输入等等
